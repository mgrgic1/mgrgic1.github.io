C++ 2D Game Engine

C++ SFML Library Game Development

A simple 2D Game Engine made using C++ and the SFML library.

This game engine gives users basic, yet fundamental, tools to make a 2D game. There is a state manager for switching between "scenes", animation logic, input manager, and asset manager (for importing art, sound, fonts, etc).

The source code can be found here.

Tip: This project is constantly under development! My next plans are to add a particle system, light system, and to further develop current features.

C++ Breakout (Brickbreaker)

C++ SFML Library Game Development

I used the tools from my 2D Game Engine to make a simple Breakout (or Brickbreaker) game in C++.

There are three levels, which have all been loaded from an XML file using the TinyXML library. Attributes are read from the XML file and then created for each level. The game finishes when the user has beat the three levels or has run out of lives.
Features music, sound effects, collision behaviors, and 6 "scenes" (3 of which are levels)

The source code can be found here.

Portal Recreation in Unity (with VR support)

C# Unity3D

This project was a mock recreation of Portal (with VR support) using C# and Unity3D.

There are three levels the player has to navigate through a series of puzzles. The game if the player has reached the end or if the player exits the game. Features include:

  • VR Support (Oculus or Google Cardboard)
  • Controller/Keyboard support
  • Portal traversal and realtime display of opposite end of portal
  • Raycasting

The source code can be found here. (group collaboration, I am listed as a contributor.
